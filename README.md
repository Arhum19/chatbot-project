
# 🚀 Gemini AI Chat: A Simple Chatbot Using Gemini API


## 📖 Description

The Gemini AI Chat project is a simple chatbot built using the Gemini API, a powerful tool for generating human-like text. This project aims to provide a basic conversational interface for users to interact with the AI model. The chatbot is designed to respond to user input and engage in a conversation, showcasing the capabilities of the Gemini API.

The chatbot is built using Node.js, HTML, CSS, and JavaScript. The project structure is organized into several folders, each containing specific files and components. The frontend is responsible for handling user input and displaying the chat interface, while the backend handles the communication with the Gemini API and generates responses.

## ✨ Features

1. 🤖 Conversational Interface: The chatbot provides a simple interface for users to interact with the AI model, allowing them to input text and receive responses.
2. 💬 AI-Generated Responses: The Gemini API is used to generate human-like responses to user input, enabling the chatbot to engage in conversations.
3. 📊 Real-Time Feedback: The chatbot provides real-time feedback to users, allowing them to see their input and the AI's response in real-time.
4. 🔒 Secure Communication: The project uses HTTPS protocol for secure communication between the frontend and backend.
5. 📁 Organized Project Structure: The project is organized into separate folders for frontend and backend, making it easy to navigate and maintain.
6. 👻 Error Handling: The project includes error handling mechanisms to ensure that the chatbot can recover from potential errors and maintain a smooth user experience.
7. 💻 Cross-Browser Compatibility: The chatbot is designed to work seamlessly across multiple browsers and devices.
8. 🤝 Scalable Architecture: The project's architecture is designed to scale with the growing demands of users, ensuring a smooth and efficient user experience.

## 📁 Project Structure

* `frontend/`
	+ `index.html`: The main HTML file for the chat interface.
	+ `styles.css`: The CSS file for styling the chat interface.
	+ `script.js`: The JavaScript file for handling user input and interacting with the Gemini API.
* `backend/`
	+ `api/`
		- `chat.js`: The Node.js file for handling chat requests and interacting with the Gemini API.
	+ `package.json`: The project's package file for managing dependencies.
* `public/`: The folder for static assets.

## ⚙️ How to Run

### Setup

1. Clone the repository: `git clone https://github.com/your-username/gemini-ai-chat.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm run start`

### Environment

* The project uses the Google Generative AI API, which requires a valid API key. Please set the `GOOGLE_API_KEY` environment variable to your API key.
* The project uses Vercel for deployment. Please set the `VERCEL_GITHUB_TOKEN` environment variable to your GitHub token.

### Build

The project uses html,css,js for building the frontend and slightly use node.js for handling api calls.

### Deploy

The project is deployed to Vercel using the `vercel dev` command.

## 🧪 Testing Instructions

1. Run the project using the `npm run start` command.
2. Open the chat interface in a web browser.
3. Input text in the chat input field and press the send button.
4. Verify that the chatbot responds with a human-like response.
5. Test the chatbot with different inputs and scenarios to ensure it is working as expected.
